Configurations and Security of  iDisconnect v 1.0.25 ®

The iDisconnect version 1.0.25 ® behind you a menu of options with enhanced security functions *. These securities include leaving iDisconnect version 1.0.25 ® Mode Invisible at runtime. So you will have to double security that no one interfere with the operation of iDisconnect version 1.0.25 ®. The iDisconnect version 1.0.25 ® also includes other common security in software, which is protection through a password. Even knowing the person getting the Invisible Mode of  iDisconnect version 1.0.25 ®, ** she will be able to interfere in its operation if you do not have the password. Below the screen options iDisconnect version 1.0.25 ® and its resources:


1) Leaving this option enabled the iDisconnect version 1.0.25 ® will start with Windows (Remember that if other options are checked, the startup iDisconnect version 1.0.25 ® will be committed).

2) If this option is checked, at the time the User close the iDisconnect version 1.0.25 ®, it will automatically be minimized tray (in Windows clock).

* 3) If the option to Invisible Mode is enabled, the iDisconnect version 1.0.25 ® will stay in run mode invisibly at the time the user clicks the button "Program Operation", otherwise, will only minimize the Tray.

* 4) When this option is enabled for the first time, the iDisconnect version 1.0.25 ® will display a window for the user to provide a password, this same screen. After registering the password iDisconnect version 1.0.25 ® will be blocked showing the Lock screen when it starts or minimize the Tray (Remember, this window will appear to enter the password only once after the user enters the password. User can manage password in Menu> Tools). Now, if this option is not enabled the iDisconnect version 1.0.25 ® will not start or will be blocked and locked, put password will still be registered, and will be used when this option is enabled.

5) At the option User defined or not you use the theme in iDisconnect version 1.0.25 ®.

6) The function of this option is to usario show a warning message before the operation chosen by the same. This message will be triggered with 5 minutes before the selected operation, and 1 minute before the selected operation. It is a security option, if the user forgets the iDisconnect  version 1.0.25 ® running by mistake.

7) Enabling this option, iDisconnect version 1.0.25 ® will sound when you click the mouse.

8) In this field the User can choose which language  iDisconnect version 1.0.25 ® will get. For the chosen language is implemented completely iDisconnect version 1.0.25 ® will be finalized after showing an alert and the User confirm. The user will see the changes when the inicior iDisconnect Version 1.0.25 ® again.

* 9) In this option the user can choose a theme for the * iDisconnect Version 1.0.25 ® . Think in visual tastes. Choose a theme that suits you.
